I'm 22 years old and I have a passion for building and designing technology that solves problems with the least amount of complexity possible.

From software to communities, I like building things that bring people together.

In March 2013, I started Present with my friend Dan Scanlon, where we are passionate about creating the best mobile video experience.

Recently, I've been focused on making iOS development easier:

  • TableAdapter, a drop in replacement for UITableViewDataSource and UITableViewDelegate, providing a model-based approach for table views
  • APIKit, a collection of protocols and extensions for Alamofire which makes requesting and receiving JSON data trivial
  • HLSKit, a library for creating HTTP Live Streaming videos
  • CameraKit, a library for easily configuring a camera
  • PresentPrePermissions, a library to manage and request access to permissions

I'm interested in creating exceptional products and experiences, using the latest and greatest technologies. If you're interested in discussing these technologies further, or hiring me as a consultant, get in touch.
